# Список правок («что не нравится»)

> Сюда пользователь складывает мелкие правки/пожелания по UX/UI, которые
> заметил, но фиксить прямо сейчас не критично. Я (LLM-ассистент) разбираю
> этот список пачкой в конце фазы.
>
> Формат записи:
> ```
> ## YYYY-MM-DD HH:MM — короткое описание
> Где: «вкладка Наука, кнопка Начать»
> Что: «текст обрезается на узких экранах»
> Хочу: «сделать в две строки или укоротить»
> Статус: 🔴 не разобрано  / 🟡 в работе  / 🟢 сделано
> ```
>
> Если правка **критичная** (баг ломает игру, кнопка не работает) —
> не сюда, а сразу мне в чат. Я починю немедленно.

---

## Шаблон для копирования

```
## 2026-MM-DD — заголовок
Где:
Что:
Хочу:
