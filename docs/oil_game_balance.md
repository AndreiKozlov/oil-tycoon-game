# Балансировка экономики — числа для кода

> Версия: 0.1 (MVP-баланс)
> Дата: 2026-05-17
> Профиль: Вариант В — быстро на старте, медленно на топе
> Все цены в игровых $ (1 игровой $ ≈ 1 реальный $ для понятности)

---

## Принципы

### Кривая прогрессии (Вариант В)
- **Уровни 1-5 (Старт):** быстрый прогресс, дофамин каждые 15-30 минут
- **Уровни 6-8 (Средняя зона):** замедление, появляется смысл планирования
- **Уровни 9-10 (Эндгейм):** медленный прогресс, появляется смысл доната

### Магические коэффициенты
- **Цена ×3.0** на каждый следующий уровень (с уровня 1 к 10)
- **Добыча ×2.2** на каждый уровень
- **Время постройки ×1.8** на каждый уровень
- **Окупаемость растёт ×1.4** на уровень (каждый апгрейд окупается медленнее)

### Целевые сроки прогрессии (для casual игрока ~30 мин/день)
- Вышка уровня 5: **~2 недели**
- Вышка уровня 8: **~2 месяца**
- Вышка уровня 10: **~4 месяца**
- Все технологии MVP пройдены: **~3 месяца**

## 1. Нефтяная вышка (10 уровней)

Основной добывающий объект. Главная прогрессия игры.

| Ур. | Цена ($) | Добыча (бар/час) | Время постройки | Энергия (кВт) | Износ/час | Окупаемость* |
|-----|----------|------------------|-----------------|---------------|-----------|--------------|
| 1 | 5 000 | 50 | 5 мин | 5 | 1.0% | ~2 часа |
| 2 | 15 000 | 110 | 15 мин | 8 | 1.0% | ~3 часа |
| 3 | 45 000 | 240 | 45 мин | 12 | 1.1% | ~5 часов |
| 4 | 135 000 | 530 | 2 часа | 18 | 1.2% | ~7 часов |
| 5 | 400 000 | 1 170 | 5 часов | 28 | 1.3% | ~10 часов |
| 6 | 1 200 000 | 2 570 | 12 часов | 42 | 1.5% | ~14 часов |
| 7 | 3 600 000 | 5 650 | 1 день | 65 | 1.7% | ~20 часов |
| 8 | 10 800 000 | 12 430 | 2 дня | 100 | 2.0% | ~28 часов |
| 9 | 32 400 000 | 27 340 | 4 дня | 150 | 2.4% | ~40 часов |
| 10 | 97 200 000 | 60 150 | 7 дней | 225 | 3.0% | ~56 часов |

*Окупаемость = время чистой работы для окупания стоимости постройки при цене $70/бар, с учётом расходов.

### Формула
```
price(N)        = 5000 × 3^(N-1)
production(N)   = 50 × 2.2^(N-1)
build_time(N)   = 5 min × 1.8^(N-1)
energy(N)       = 5 + round(3 × 1.5^(N-1))
wear_rate(N)    = 1.0 + 0.2 × (N-1)  [% в час]
```

### Что это даёт игроку
- **Уровни 1-3:** новичок может построить за 1-2 часа игры. Быстрый дофамин.
- **Уровни 4-5:** уже есть смысл копить. Достигаются за 1-2 недели casual игры.
- **Уровни 6-7:** месяц-два целенаправленного развития.
- **Уровни 8-10:** эндгейм. Здесь донат становится осмысленным (ускорить постройку).

## 2. Газовая скважина (7 уровней — Expansion content, в MVP 5 уровней)

|-----|----------|---------------------|-----------------|---------------|-----------|

| 2 | 60 000 | 2.5 | 1.5 часа | 15 | 0.8% |
| 3 | 180 000 | 6.0 | 4 часа | 25 | 0.9% |
| 4 | 540 000 | 15.0 | 12 часов | 40 | 1.0% |

| 6 | 4 800 000 | 80.0 | 2 дня | 90 | 1.4% |

```
price(N)       = 20000 × 3^(N-1)
```

### Особенности газа

- **Требует газопровод** для доставки к резервуарам/продаже
- Цена на бирже **более волатильная** (зимой ×1.5-2.0)

## 17. Формулы (для генерации остального контента)

Эти формулы можно прямо в код вставлять.

### Цены постройки/апгрейда
```javascript
function buildCost(baseCost, level) {
  return baseCost * Math.pow(3.0, level - 1);
}

function buildTime(baseTimeMin, level) {
  return baseTimeMin * Math.pow(1.8, level - 1);
}
```

### Добыча
```javascript
function production(baseProd, level) {
  return baseProd * Math.pow(2.2, level - 1);
}
```

### Расход энергии
```javascript
function energyConsumption(level) {
  return 5 + Math.round(3 * Math.pow(1.5, level - 1));
}
```

### XP для уровня
```javascript
function xpRequired(playerLevel) {
  if (playerLevel <= 1) return 0;
  return Math.round(1000 * Math.pow(2.0, playerLevel - 2));
}
```

### Окупаемость (в часах работы)
```javascript
function paybackHours(buildingCost, productionPerHour, sellPricePerBarrel, costsPerHour) {
  const incomePerHour = productionPerHour * sellPricePerBarrel - costsPerHour;
  if (incomePerHour <= 0) return Infinity;
  return buildingCost / incomePerHour;
}
```

### Цена ускорения через премиум
```javascript
function speedUpCost(remainingTimeMin) {
  // Базовая ставка: 1 кристалл за 5 минут оставшегося времени, с дисконтом
  const base = Math.ceil(remainingTimeMin / 5);
  const discount = remainingTimeMin > 60 ? 0.8 : 1.0;
  return Math.max(1, Math.floor(base * discount));
}
```

### Регенерация цены при колебаниях биржи
```javascript
function gamePrice(realPrice, eventMultiplier, microFluctuation) {
  return realPrice * eventMultiplier * microFluctuation;
}
```
