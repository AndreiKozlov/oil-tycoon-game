## 📋 Краткое описание игры (для контекста)

**Название (рабочее):** Oil Tycoon
**Жанр:** Симулятор-стратегия добычи нефти и газа
**Платформа:** Telegram Mini App + Web (старт), iOS + Android (потом)
**Целевая аудитория:** 20-45 лет, играют в Forge of Empires / Township / Travian
**Уникальные фишки:**
1. Реальные биржевые цены на нефть (Brent/WTI/Urals/Dubai/Heavy)
2. Реальные новости нефтегаза триггерят игровые события (OPEC сократил → +30% к цене в игре)
3. 5 типов нефти из разных регионов мира (стратегический выбор)
4. Эпохи по 90 дней с тематическими картами (Сибирь → Аравия → Северное море → ...)

**Core loop:**
Разведка участка → бурение → постройка вышки → добыча → продажа на бирже → апгрейд оборудования → исследование технологий → расширение

**Монетизация:**
Премиум-валюта (кристаллы) → ускорение построек, бустеры, скины, +слоты участков. Никакой pay-to-win.

**Конкуренты:**
Hamster Kombat (кликер, мёртв) / Notcoin (кликер, мёртв) / Catizen (кликер) / Oil Tycoon мобильный (наш ближайший) / Forge of Empires (направление куда растём)

**Наше отличие:** глубокий симулятор с реальной экономикой, не очередной тапалка.

## ⚠️ Что НЕ предлагать пользователю (триггер-темы)

Если пользователь спросит или предложит — мягко вернуть к плану:

1. **«А может токен на TON прямо сейчас?»**
   → Нет. Юридические риски + $45k бюджета не на что. Только этап 6+, если будет выручка.

2. **«А давай сразу мультиплеер?»**
   → Нет. Сначала прототип-одиночка (3-4 недели) — проверить что механика интересна. Иначе 6 месяцев пилим сервер для скучной игры.

3. **«А Apple/Google нам дадут буст в магазине»**
   → Нет, миф. Apple/Google не пропустят игру с криптой, требуют юрлицо $5-15k, бан за нарушения. Telegram сначала, App Store потом.

4. **«А давай ресурсы кончатся навсегда, как на Земле»**
   → Нет. Это убьёт игру если она выстрелит. Эпохи решают: дефицит в эпохе → новая карта раз в 90 дней.

5. **«А давай биржу внутри игры где игроки торгуют за реальные деньги»**
   → Нет. Это финансовое посредничество без лицензии (ст. 172 УК в РФ, MiCA в ЕС). Только внутриигровая валюта.

6. **«А давай я отправлю сообщение через Telegram-бот»**
   → У меня нет инструмента для отправки сообщений в Telegram. Только Bash, Read, Edit, Write в этом окружении.
