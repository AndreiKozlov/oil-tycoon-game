## Дизайн (зафиксировано в START_HERE.md, дублирую сюда)

- Жанр: симулятор-стратегия (НЕ кликер).
- Платформы: Telegram Mini App + Web → потом App Store / Google Play.
- Без токена/крипты на старте. Только этап 6+, если игра выстрелит.
- Эпохи по 90 дней, стартовая «Сибирская» (30 000 участков).
- Сессия 10-15 мин активной, 4-6 раз/день.
- Геолог Иван — наставник в туториале.
- Реальные биржевые цены (Yahoo Finance: Brent/WTI/Urals/Dubai/Heavy).
- Реальные новости из RSS триггерят игровые события.
- UX продажи: только итог, без расчётов.
- Прогрессия: быстро на старте, медленно на топе.

- Фронт: TypeScript + React 18 + Vite + Pixi.js 8 + Tailwind + shadcn/ui + Lucide.

## 2026-05-18 — БОЛЬШОЙ ПОВОРОТ: новая концепция игры (v2)

Этапы A–F (нефтяной прототип) **законсервированы в ветке `legacy-oil-only`** на GitHub. Игрался, работал в Telegram, прошёл D.O.D. этапа 1. Сохранён как референс/страховка.

Новая концепция (этапы G+):
- **Жанр:** стратегия с глобальной картой, не симулятор одной шахты.
- **Карта:** **30 000 тайлов**, общая для всех игроков (не «у каждого своя»).
- **Биом тайла** определяет ресурс: лес / камень / руда / нефть / газ / плодородная земля / пустыня / вода / горы. Часть тайлов бесресурсные — на них только инфраструктура (город, НПЗ, дороги).
- **Orientation:** landscape (полный поворот). Mobile-first для landscape: tilemap в центре, узкие панели по краям.
- **Tilemap:** изометрические SVG-спрайты (один спрайт-шит). Стиль — плоский, читаемый. «Playrix-Township» отложен (требует ручного 3D-арта на этапе 2+).
- **Стартовый участок** даётся **случайно** при первом входе. Все остальные участки **покупаются за игровые деньги** (премиум-ускорение через кристаллы). Pay-only-progress отклонён.
- **Разведка** перед покупкой: показывает запасы с **погрешностью**, которая уменьшается по мере прокачки технологии разведки (geo-1 = ±50%, geo-5 = ±5%).
- **Заброшенные участки:** если игрок не заходил **14 дней** (значение настроим по DAU), участок помечается «заброшенным», другие могут купить дешевле, но с поломанным оборудованием. Требует бэкенда.
- **Расчёт на 10k+ DAU** — проектируем под нагрузку с первого дня: кеш-карты в Redis, шардирование на регионы, эндпоинты возвращают только видимые тайлы.
- **Ресурсы:** лес, камень, железная руда, еда, нефть, газ, электричество. Производные: пиломатериалы, стройматериалы, бензин, дизель.

## 2026-05-17 — HTTPS для Telegram через Cloudflare Quick Tunnel (временно)

- Туннель работает в обычном вебе и в Telegram. URL вида `*.trycloudflare.com`.
- **В РФ trycloudflare.com периодически режется РКН/провайдерами.** Пользователь подтвердил это — открыл через прокси, получилось.
- Пока **остаёмся на Cloudflare для разработки**: тестируем сами через прокси/VPN. Для бета-теста (этап 3) перейдём на нормальный домен (саб от tradearena.tech или новый) + Let's Encrypt.
- Не предлагать сменить HTTPS-схему до явной просьбы или старта беты.
